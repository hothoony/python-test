### python 가상환경 설정 (vscode)
- command + shift + P
  - Python: Create Environment 선택
  - Venv 선택
  - Python installation 선택

# 가상환경 생성
python3 -m venv .venv

### python 가상환경 활성화
```bash
source .venv/bin/activate
```

### python 패키지 설치
```bash
pip install pycryptodome
```

### python 패키지 삭제
```bash
pip uninstall pycryptodome
```

### python 패키지 목록
```bash
pip list
```
