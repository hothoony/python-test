## python 디렉토리 구조
```shell
my_project/
├── Makefile              # 작업 자동화
├── requirements.txt      # 의존성 목록
├── src/                  # 소스 코드
│   └── __init__.py
│   └── main.py
├── tests/                # 테스트 코드
│   └── __init__.py
│   └── test_main.py
├── .gitignore            # Git 제외 파일
├── README.md             # 프로젝트 설명
└── .env                  # 환경 변수 (선택)
```
